# WebCraft Studio — Modern Responsive Web Practice

WebCraft Studio adalah refactor ringan dari repository praktikum web lama menjadi project portfolio frontend dasar yang tetap sederhana, cepat dibuka secara lokal, dan mudah dikembangkan.

Project ini mempertahankan konsep halaman landing praktikum, tetapi struktur folder, HTML, CSS, dan JavaScript-nya sudah dirapikan agar lebih semantik, responsif, dan mudah dibaca.

## Teknologi

- HTML5 semantik
- CSS3 dengan variabel, layout modern, dan responsive breakpoints
- JavaScript vanilla untuk navigasi mobile, tahun otomatis, dan validasi form
- Asset lokal tanpa framework berat

## Cara Menjalankan

1. Buka file `index.html` langsung di browser, atau gunakan live server dari VS Code.
2. Tidak ada proses build dan tidak ada backend.
3. Jika ingin melihat versi lama, folder `page-1/` dan `FIGMA/` tetap disimpan sebagai referensi.

## Struktur Inti

- `index.html` halaman utama
- `css/style.css` gaya dasar
- `css/responsive.css` penyesuaian tablet dan mobile
- `js/main.js` interaksi ringan
- `assets/` gambar dan ikon
- `docs/project-structure.md` dokumentasi struktur dan perubahan
- `screenshots/` tempat menyimpan screenshot dokumentasi

## Catatan

- Project ini tetap ringan dan bisa dibuka secara lokal.
- Tidak menggunakan React, Vue, Angular, Next.js, atau backend.
- Asset penting dari versi lama dipertahankan dan dipindahkan ke struktur yang lebih jelas.
